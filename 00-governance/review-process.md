# Review Process

The review process converts AI-generated candidates into SME-reviewed modernization knowledge.

## Steps

1. Intake candidate legacy information from indexed source evidence.
2. Generate a Markdown SME Review Pack.
3. Review the pack with the SME using GitHub Copilot Chat.
4. Capture SME choices and corrections in a reviewed Markdown pack.
5. Apply accepted or corrected results to JSON dictionaries and mapping candidates after review.
6. Report open questions, conflicts, and review status.

## SME Review Rules

- Ask one item at a time.
- Show the AI Recommended Answer, evidence, confidence, and A/B/C/D/E options.
- Accept either a choice or natural language correction.
- Keep SME questions focused on legacy understanding.
