# AI Output Review Standard

AI output must be reviewable, traceable, and easy for SMEs to correct.

Each review item must include:

- Item ID
- Type
- Legacy object
- AI Recommended Answer
- Why AI recommends this
- Evidence
- Confidence
- Alternative Answers
- Question for SME
- Options A/B/C/D/E
- SME Selected Option
- SME Corrected Answer
- SME Comment
- Review Status

AI should not invent evidence. Missing or weak evidence should result in low confidence.
