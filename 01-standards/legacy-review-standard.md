# Legacy Review Standard

Legacy review focuses on understanding the current IBM iSeries / AS400 environment.

Use legacy terms such as program, file, field, job, report, screen, and batch flow.

Review questions should confirm:

- What the legacy object is
- What business or technical purpose it serves
- Where the evidence came from
- Whether the AI interpretation is accepted, corrected, rejected, or still open

Do not ask the SME to approve target architecture, service decomposition, API design, or Java implementation details.
