# Mappings

This directory contains machine-readable JSON mapping candidates between legacy objects and target implementation objects.

Mappings are candidates until reviewed and approved. Do not treat unreviewed mappings as final.

Current implementation note: these files are reserved candidate surfaces. No
current script generates or applies mapping candidates.

Current mapping files:

- `legacy-program-to-target-service.json`
- `legacy-file-to-target-table.json`
- `legacy-field-to-target-field.json`
