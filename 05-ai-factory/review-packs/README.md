# Review Packs

This directory contains Markdown SME Review Packs for optional selective review.

Review packs are not the default MVP workflow. The default workflow is Auto Wiki
Intake into `03-wiki/`.

Use review packs only for low-confidence items, conflicts, critical programs,
key business fields, and important open questions.

The SME should review one item at a time and answer with `A`, `B`, `C`, `D`, or
`E`, or provide a natural language correction.

Only explicit SME action may produce `SME Confirmed`.
