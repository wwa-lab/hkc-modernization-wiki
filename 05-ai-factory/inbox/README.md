# Inbox

Place small local sample files here when running Auto Wiki Intake without a full
intake manifest.

The real LAN folder remains the raw document source of truth. This inbox is only
for controlled local samples, extracts, or references that are safe to store in
the repo.

Windows example:

```powershell
py -3 05-ai-factory\scripts\process_intake.py --inbox 05-ai-factory\inbox
```
